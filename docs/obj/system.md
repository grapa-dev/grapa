# From $OBJ

## exec(op)
Executes string in grapa.

```
> "4*5".exec()
20
```

## shell()
Executes string in the shell of the operating system. 

```
> "ver".exec()
Microsoft Windows [Version 10.0.18363.778]
```

## post()
Sends string to console input, processing as if it came in from the console.
